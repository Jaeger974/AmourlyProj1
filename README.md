# Amourly Subscription Service

A dynamic web application built with Node.js, Express, Javascript, and EJS, Google, facebook and Twitter OAuthentication, Postgres database usage, password salting and hashing, cookie sessions, featuring static assets (CSS, JS, images), modular views. This project demonstrates clean architecture, responsive design, and scalable templating.

All code and structure has been authored by myself or through course-based learning and element sampling. 

Amourly is a subscription-based love letter service designed for modern relationships. Amourly was born from a simple problem: meaningful romantic gestures take time, and life doesn’t always allow for it. Finding the perfect poem, writing it beautifully, sending it… and hoping it arrives? Not exactly practical.

Amourly solves this.

✨ Service Features
- Scheduled poem delivery — choose how often you want to show your love
- Personalised content — tailor poems to your partner’s interests (from Jane Eyre to Shakespeare to Lord of the Rings)
- Effortless romance — no calligraphy skills or post office trips required.
